# Standard_curve_in_julia
An Pluto notebook for the Create of Standard curves in julia

This is a Pluto.jl notebook

Instructions on how to use:
1. Download Julia: https://julialang.org/downloads/manual-downloads/
2. Open the terminal (for linux) or the command prompt in windows..
3. Type julia
4. Type ]
5. Type add Pluto
6. Press the backspace key
7. Type import Pluto
8. Type Pluto.run(), this would open a web browser tab, use the program there.
9. Download and run the standard_curve.jl on the opened tab.
